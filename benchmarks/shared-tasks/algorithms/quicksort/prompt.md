Implement quicksort in Python.

Requirements:
1. Function signature: def quicksort(arr: list) -> list
2. Use the quicksort algorithm with proper partitioning
3. Return a new sorted list (don't modify input)
4. Use recursive approach with divide-and-conquer strategy
5. Include type hints and docstring
6. Handle edge cases (empty list, single element, duplicates)

Example:
>>> quicksort([3, 6, 8, 10, 1, 2, 1])
[1, 1, 2, 3, 6, 8, 10]

>>> quicksort([])
[]

>>> quicksort([42])
[42]

Time complexity should be O(n log n) average case.
