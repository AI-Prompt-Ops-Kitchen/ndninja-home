Implement quicksort in Python.

Requirements:
1. Function signature: def quicksort(arr: list) -> list
2. Use the quicksort partitioning algorithm
3. Do not modify the input list (return a new sorted list)
4. Handle edge cases (empty list, single element, duplicates)
5. Include type hints and docstring

Example:
>>> quicksort([3, 1, 2])
[1, 2, 3]

Time complexity should be O(n log n) average case.
