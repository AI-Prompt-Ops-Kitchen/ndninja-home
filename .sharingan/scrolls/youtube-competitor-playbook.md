## The Viral Formula: Deconstructed

### "Theresa in Endfield?!" — 229K views, 15 seconds

**Structure (second by second):**
1. **0-2s:** Game rating/logo screen (sets context, builds anticipation)
2. **2-6s:** Slow cinematic pan UP from character's feet → boots → legs (classic waifu reveal technique)
3. **6-10s:** Continue pan up through dress/outfit (tension building, viewer locked in)
4. **10-12s:** Full character face reveal + bold "NEW" text overlay
5. **12-14s:** Meme reaction GIF composited in corner (guy in suit losing his mind)
6. **14-15s:** Final character art beauty shot

**Why it worked:**
- **Anticipation mechanics:** The slow pan forces viewers to watch the full reveal. You can't skip — you NEED to see the face.
- **Emotional proxy via meme:** The reaction GIF tells viewers HOW to feel. It's social proof in 2 seconds ("this is so hype even THIS guy is freaking out").
- **Ultra-short loop:** At 15s, it auto-loops on Shorts. Viewers watch 2-3x without realizing, which juices the algorithm.
- **Search + emotion title:** "Theresa in Endfield?!" combines a searchable character name with emotional punctuation.
- **Bilibili source credit:** Used high-quality fan art/animation from Chinese creators, properly credited.

### The Meme Reaction Overlay Pattern
This is the technique the user specifically noticed. It works because:
- Replaces the need for a voiceover or talking head
- Universal emotional language (everyone understands a "freaking out" GIF)
- Adds humor/personality to raw game footage
- Extremely low effort to produce (game clip + GIF overlay + text)
- Creates shareability ("lmao this is exactly how I reacted")

## Content Archetypes That Work (Ranked by ROI)

### Tier S — Viral Potential (High views, minimal effort)
1. **Waifu/Husbando Reveal + Meme Reaction** — Slow character reveal with reaction GIF overlay. 15s max.
2. **Community Meme Shitpost** — Inside joke content for specific game fandoms. Shared in Discord/Reddit.
3. **"Which Would You Pick?" Engagement Bait** — Binary choice + game footage. Drives comments.

### Tier A — Consistent Views (Search-driven)
4. **Lore Moment Clips** — Key story moments clipped to the emotional peak. Descriptive titles for search.
5. **First Appearance Clips** — "[Character] First Appearance | [Game] [Version]" — evergreen search content.

### Tier B — Brand Building (Lower views, higher loyalty)
6. **Hot Take Commentary** — What we already do. Avatar + opinion + B-roll.
7. **News Roundup** — Multi-topic coverage. Good for regulars, harder to go viral.

### Tier C — Low ROI
8. **Raw Gameplay** — Unedited game footage. Only works for brand-new releases.
9. **Long Cutscene Uploads** — Diminishing returns past 60 seconds.

## Tag Strategy (Stolen From RaumMkII)

Every video should have **15-22 tags** covering:
- English game title + abbreviation (e.g., "Wuthering Waves", "WuWa")
- Japanese game title (e.g., "鳴潮")
- Character names (English + Japanese/Chinese)
- Version number (e.g., "WuWa 3.1", "version 3.1")
- Content type keywords ("gameplay", "cutscene", "meme", "reaction")
- Generic discoverability ("shorts", "fyp", "gaming")
- Hashtag versions in description AND tags

**Template:**
```
[game name], [game abbreviation], [game japanese], [character name],
[character name japanese], [game version], [game] gameplay,
[game] cutscene, [game] memes, [game] reaction, gaming, shorts, fyp,
[game] [version] [content type], [relevant trending hashtag]
```

### Winning Strategy: The Hybrid Approach
The data says: Use our avatar/dual-anchor format NOT for straight news delivery, but wrapped in the engagement FORMATS that the algorithm rewards:
- **"Ninja Rates"** — Adapt IMGWatcher's overlay format with our avatar

- **Quick Reaction Clips** — ChisaCalamity speed: sub-30s avatar reactions posted within HOURS of reveals
- **Community Shitposts** — urgigacatto energy: insider memes featuring our characters
- **Soft Conversational Titles** — NoPengoo proves these outperform ALL-CAPS clickbait in gacha

**The news commentary is the DIFFERENTIATOR (nobody else does it), but it must be wrapped in engagement formats the algorithm rewards.**

## Secret Sauce Formula (Working Theory)

```
VIRAL SHORT = (Trending Game Moment × Emotional Hook × Meme Layer)
              ÷ Duration
              + Community Insider Energy
              + Search-Optimized Tags
```

Components:
- **Trending Game Moment:** New character, update, controversy, reveal
- **Emotional Hook:** Anticipation (slow reveal), hype (reaction), humor (meme), debate (choice)
- **Meme Layer:** Reaction GIF, bold text, sound effect, or avatar expression
- **Duration:** Shorter = higher view velocity. 15s > 30s > 60s for virality
- **Community Energy:** Inside jokes, fandom references, "if you know you know"
- **SEO Tags:** 15-22 multi-language tags covering character + game + version

**Our unique multiplier:** We can replace generic meme GIFs with our own animated Ninja/Glitch reactions — making the meme layer BRANDED. Nobody else has this.
